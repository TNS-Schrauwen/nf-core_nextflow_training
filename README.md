# A Guide to Nextflow & nf-core

This beginner's guide introduces you to the fundamentals of Nextflow and the nf-core ecosystem, giving you the tools to build, run, and contribute to reproducible bioinformatics pipelines. By the end of the session, you will have run your first Nextflow workflow, explored a real nf-core pipeline, and taken your first steps toward contributing to a 
global open-source community. Happy Nextflow-ing! 🚀🎉

## Prerequisites

- Basic UNIX command line skills
- A GitHub account (for GitHub Codespaces)
- A modern web browser (Chrome, Firefox, Safari)

## Viewing the Presentation

1. Clone the repository:
```bash
   git clone https://github.com/TNS-Schrauwen/nf-core_nextflow_training.git
```

2. Open the `training.html` file in your browser by double-clicking it.
